# abarraza24-web-project

## Project 2 - HTML Foundation Page

For Project 2, I created a simple website about nutrition and bodybuilding. I picked this topic because fitness is a real part of my life, and I wanted my page to reflect something personal instead of choosing a random subject.

While creating this page, I learned more about how to structure an HTML document with headings, paragraphs, links, images, navigation, and tables. One challenge I faced was making the page look organized using only HTML inline styling without using CSS.


## Project 3 - External CSS Styling Page

For Project 3, I used my Project 2 nutrition and bodybuilding page as the foundation and upgraded it with an external CSS stylesheet. The page still focuses on calories, macros, training, and why fitness connects to my personal goals.

While creating this page, I learned how to connect an external CSS file to an HTML page using the link element. I also practiced the CSS box model from Zybooks 4.7 by using margin, padding, borders, width, and centered containers. This helped me understand how spacing works around different HTML elements.

One challenge I faced was moving the inline styles from Project 2 into a separate stylesheet while keeping the page organized and visually consistent. I also practiced using relative paths for navigation, images, and the stylesheet.
